------------------Проект "Змійка" (курсова робота------------

Програма призначена для комп'ютерів, що працюють під керівництвом MacOS X. На Windows платформі код потребує деяких доповнень.

Дана гра являє собою розширений варіант класичної змійки. Мета гри - утримати змію в межах ігрового поля і від перетину власного тіла. Час від часу на полі з'являються призи і пастки (бомби). Змія голодна і не проти підкріпитися призами, якщо змія з'їсть приз, то її тіло виростає на одну секцію. Пастки ж змія повинна уникати - якщо змія потрапляє в пастку, то її тіло зменшуєтсья на одну секцію.

Гру було розроблено у середовищі Xcode під управлінням Mac OS X з використанням мультимедійної бібліотеки SFML.

До проекту додається коротке відео роботи програми (геймплей).

Робочий каталог (working directory) - ця папка (TropicalSnake).
